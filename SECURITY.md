# セキュリティポリシー

## 脆弱性の報告

セキュリティ上の問題を見つけた場合は、**公開の Issue を作成せず**、
GitHub の **Private vulnerability reporting**（本リポジトリの "Security" タブ →
"Report a vulnerability"）からご報告ください。

- できるだけ再現手順・影響範囲・想定される深刻度を添えてください。
- 受領後、状況の確認と対応方針について折り返しご連絡します。
- 修正がリリースされるまで、詳細の公開はお控えいただけると助かります。

## サポート対象

原則として `main` ブランチ（最新リリース）を対象に対応します。
